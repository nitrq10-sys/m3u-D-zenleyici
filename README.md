# m3u-D-zenleyici
M3u Kanal Listenizi Düzenleyin

Nasıl kullanılır

V1

HTML dosyasını indirip tarayıcıda aç

.m3u dosyasını yükle (dosyayı sürükle-bırak da çalışır)

Sol sütun kanal listesi — isme göre ara, gruba göre filtrele

Sağ sütun + ile eklenen kanallar

Tümünü Ekle ile filtrelenmiş kanalları toplu ekle

Sağ sütunda kanalları sürükleyerek sıralayabilirsin

💾 .m3u Olarak Kaydet ile yeni listeyi indir

17.000 kanallı liste için optimize edilmiş — arama yapınca performans sorunsuz çalışır.

V2

Güncellendi! Yapılan değişiklikler:

Sayfa tam ekrana sığıyor — kanal listeleri sayfayı taşırmak yerine kendi içinde kaydırılıyor

💾 Kaydet — hızlı kayıt, secilen_kanallar.m3u olarak indirir

📝 Farklı Kaydet — açılan modal'dan dosya adı yazıp kaydedebilirsin (Enter ile de çalışır)
